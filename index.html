HTML Tags and Their Uses

Seperated by Category

Basic HTML
Tag Name Description
<!DOCTYPE> Document Type Tag Defines the type of document
<html> HTML Tag Defines a document as HTML. All html code should be written within
<head> Head Tag Contains metadata/information for the document (including title)
<title> Title Tag Defines a title for the document. Will be placed on the web browser tab
<body> Body Tag Defines the body of the document
<h1> to <h6> Heading Tags Used to differentiate headings. <h1> is largest. <h6> is smallest
<p> Paragraph Tag Defines a paragraph
<br> Line Break Tag Inserts a line break
<hr> Horizontal Rule Tag Defines a thematic change in the content
<!---...--> Comment Tag Defines a comment

Formatting
Tag Name Description
<abbr> Abbreviation Tag Defines an abbreviation or acronym
<address> Address Tag Defines contact information for the author/owner of a document/article
<b> Bold Tag Defines bold text
<bdi> Bi-Directional Isolation Tag Isolates a part of text that might be formatted in a different direction from other text outside it
<bdo> Bi-Directional Override Tag Overrides the current text direction
<blockquote> Quote Tag Defines a section that is quoted from another source
<cite> Cite Tag Defines the title of a work
<code> Code Tag Defines a piece of computer code
<del> Deleted Tag Defines text that has been deleted from a document
<dfn> Define Tag Specifies a term that is going to be defined within the content
<em> Emphasize Tag Defines emphasized text
<i> Italicize Tag Defines a part of text in an alternate voice or mood
<ins> Insterted Tag Defines a text that has been inserted into a document
<kbd> Keyboard Tag Defines keyboard input. Will cause the element to appear as a key on the web page
<mark> Marked Tag Defines marked/highlighted text
<meter> Scale Tag Defines a scalar measurement within a known range (a gauge)
<pre> Preformatted Tag Defines preformatted text
<progress> Progress Tag Represents the progress of a task
<q> Short Quote Tag Defines a short quotation
<rp> Ruby Parenthesis Tag Defines what to show in browsers that do not support ruby annotations
<rt> Ruby Typography Tag Defines an explanation/pronunciation of characters (for East Asian typography)
<ruby> Ruby Annotation Tag Defines a ruby annotation (for East Asian typography)
<s> Spellcheck Tag Defines text that is no longer correct
<samp> Sample Tag Defines sample output from a computer program
<small> Small Tag Defines smaller text
<strong> Strong Tag Defines Important Text
<sub> Subscript Tag Defines subscripted text
<sup> Superscript Tag Defines superscripted text
<template> Template Tag Defines a container for content that should be hidden when the page loads
<time> Time Tag Defines a specific time (or datetime)
<u> Unarticulated Tag Defines some text that is unarticulated and styled differently from normal text
<var> Variable Tag Defines a variable
<wbr> Word Break Opportunity Tag Defines a possible and specifies where in a text it would be ok to add a line-break

Forms and Input
Tag Name Description
<form> Form Tag Defines an HTML form for user input
<input> Input Tag Defines an input control
<textarea> Text Area Tag Defines a multiline input control (text area)
<button> Button Tag Defines a clickable button
<select> Selection Tag Defines a drop-down list
<optgroup> Related Options Tag Defines a group of related options in a drop-down list
<option> Options Tag Defines an option in a drop-down list
<label> Label Tag Defines a label for an input element
<fieldset> Fieldset Tag Groups related elements in a form
<legend> Legend Tag Defines a caption for a fieldset element
<datalist> Data List Tag Specifies a list of pre-defined options for input controls
<output> Output Tag Defines the result of a calculation

Frames
Tag Name Description
<iframe> Inline Frame Tag Defines an inline frame

Images
Tag Name Description
<img> Image Tag Defines an image
<map> Image Map Tag Defines a client-side image map
<area> Image Map Area Tag Defines an area inside an image map
<canvas> Canvas Tag Used to draw graphics, on the fly, via scripting (usually JavaScript)
<figcaption> Figure Caption Tag Defines a caption for a figure element
<figure> Figure Tag Specifies self-contained content
<picture> Picture Frame Tag Defines a container for multiple image resources
<svg> SVG Tag Defines a container for SVG graphics

Audio/Video
Tag Name Description
<audio> Audio Tag Defines sound content
<source> Source Tag Defines multiple media resources for media elements (video, audio, and picture)
<track> Track Tag Defines text tracks for media elements (video and audio)
<video> Video Tag Defines a video or movie

Links
Tag Name Description
<a> Hyperlink Tag Defines a hyperlink
<link> Link Tag Defines the relationship between a document and an external resource (most used to link to style sheets)
<nav> Navigation Tag Defines navigation links

Lists
Tag Name Description
<ul> Unordered List Tag Defines an unordered list
<ol> Ordered List Tag Defines an ordered list
<li> List Item Tag Defines a list item
<dl> Description List Tag Defines a description list
<dt> Description List Term Tag Defines a term/name in a description list
<dd> Description Describer Tag Defines a description of a term/name in a description list

Tables
Tag Name Description
<table> Table Tag Defines a table
<caption> Table Caption Tag Defines a table caption
<th> Table Header Cell Tag Defines a header cell in a table
<tr> Table Row Tag Defines a row in a table
<td> Table Data Tag Defines a cell in a table
<thead> Table Head Tag Groups the header content in a table
<tbody> Table Body Tag Groups the body content in a table
<tfoot> Table Footer Tag Groups the footer content in a table
<col> Column Tag Specifies column properties for each column within a colgroup element
<colgroup> Column Group Tag Specifies a group of one or more columns in a table for formatting

Styles and Semantics
Tag Name Description
<style> Style Tag Defines style information for a document
<div> Divider Tag Defines a section in a document
<span> Span Tag Defines a section in a document
<header> Header Tag Defines a header for a document or section
<footer> Footer Tag Defines a footer for a document or section
<main> Main Content Tag Specifies the main content of a document
<section> Section Tag Defines a section in a document
<article> Article Tag Defines an article
<aside> Aside Tag Defines content aside from the page content
<details> Details Tag Defines additional details that the user can view or hide
<dialog> Dialog Box Tag Defines a dialog box or window
<summary> Summary Tag Defines a visible heading for a details element
<data> Data Tag Adds a machine-readable translation of a given content

Meta Info
Tag Name Description
<head> Head Tag Defines information about the document
<meta> Metadata Tag Defines metadata about an HTML document
<base> Base Tag Specifies the base URL/target for all relative URLs in a document

Programming
Tag Name Description
<script> Script Tag Defines a client-side script
<noscript> Unsupported Script Tag Defines an alternate content for users that do not support client-side scripts
<embed> Embed Tag Defines a container for an external (non-HTML) application
<object> Embedded Object Tag Defines an embedded object
<param> Parameter Tag Defines a parameter for an object
